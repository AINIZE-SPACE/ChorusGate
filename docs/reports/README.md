# ChorusGate 迭代报告 (reports/)

按迭代组织。命名规则：`sprint-N-<类型>-<作者>.md`

| 类型 | 格式 | 说明 |
|------|------|------|
| 日报 | `sprint-N-daily-<author>.md` | 每日进展，按作者分 |
| 迭代回顾 | `sprint-N-retrospective.md` | 迭代级回顾总结 |
| 角色复盘 | `sprint-N-retrospective-roles.md` | 按角色视角复盘 |

| 作者标识 | 对应 |
|----------|------|
| `claude` | 小克 (Claude Code) |
| `hermes` | 小马 (Hermes) |

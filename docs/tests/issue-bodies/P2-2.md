## 问题
三方配置文档不一致：
- `.claude/mcp.json` (实际部署配置)
- `.claude/mcp.json.example` (示例模板)
- `docs/feature-mcp-server.md` (文档说明)

三处的 server 列表、env 字段、命令格式各不一样，无单一 source of truth。新用户照文档抄配置可能与实际/example 都对不上。

## 修法
引入单一 source of truth。两种思路：
- **(A)** 从代码生成 `.claude/mcp.json.example`（避免手维护漂移）
- **(B)** 在 `docs/feature-mcp-server.md` 加明确声明「以 `.claude/mcp.json.example` 为准」，并加 lint 在 CI 校验三者字段一致

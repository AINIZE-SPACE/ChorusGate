## 问题
`docs/feature-mcp-server.md`「MCP Tools」表只列 7 个工具，但实际 `src/index.ts` 工具数组注册 8 个。

**文档表（7 个）：** reply, send_message, add_reaction, channel_history, thread_replies, list_channels, get_user_info

**实际 src/index.ts tools 数组（8 个）：** replyTool, sendMessageTool, addReactionTool, channelHistoryTool, threadRepliesTool, listChannelsTool, getUserInfoTool, **getSkillListTool**

PR #39 diff 中 `+import { getSkillListTool } from "./tools/get-skill-list.js"` 和 `+  getSkillListTool,` 确认第 8 个工具是本 PR 新加，但 doc 表未更新。

## 影响
- 文档与实现漂移
- 用户/开发者查文档时漏掉一个工具能力
- STORY-9 收口强调"文档一致"，本项与收口目标相悖

## 修法
在 `docs/feature-mcp-server.md` 工具表加一行 `slack_get_skill_list`（或工具实际名 + 简短说明）。

## 验收
- `src/index.ts` tools 数组长度 = `docs/feature-mcp-server.md` 工具表行数
- 每个 tools 数组条目在 doc 表中都有对应行
